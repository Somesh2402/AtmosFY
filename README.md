# AtmosFY
🌈 AtmosFY – A dynamic weather app with scroll-triggered color shifts, real-time AQI, 7‑day forecast, and 14‑day trend charts. Built with Flask + OpenWeatherMap API. Single-file deployment.



🌈 AtmosFY – Chroma Weather App

*Live weather intelligence with a psychedelic twist.*  
Scroll down and watch the background morph through deep purples, electric blues, and sunset corals – powered by real atmospheric data.

## ✨ Features

- 🔍 *Search any city* or *auto‑detect your location* (geolocation)
- 🌡️ *Current conditions*: temperature, feels‑like, humidity, wind, visibility, pressure, dew point
- 🌅 *Sunrise & sunset* times
- 🍃 *Air Quality Index (AQI)* with colour‑coded labels (Good → Very Poor)
- 📅 *7‑day forecast* with daily high/low, humidity, and weather icons
- 📈 *14‑day trend chart* (temperature + humidity) using Chart.js
- 🎨 *Scroll‑activated colour transition* – the UI smoothly shifts hues as you scroll
- 📱 *Fully responsive* – works on desktop, tablet, and mobile
- 🚀 *Single‑file Flask app* – no complex folder structure, just run app.py
